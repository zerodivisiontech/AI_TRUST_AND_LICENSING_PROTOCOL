# Contributing to the AI Trust & Licensing Protocol (ATLP)

Thank you for your interest in contributing to the **AI Trust & Licensing Protocol (ATLP)**.  
This repository contains *public-safe documentation only*.  
All internal architecture, algorithms, enforcement logic, and trust scoring remain private
under a patent-pending master specification.

This document outlines how you can safely and effectively contribute.

# üìù How to Contribute

1. **Open an Issue**  
   - For suggestions  
   - Corrections  
   - Public-safe proposals  
   - Documentation improvements  

2. **Submit a Pull Request (PR)**  
   - PRs should modify *only* public documentation  
   - PRs must be well-structured and reference the issue they address  

3. **Sign the CLA (Contributor License Agreement)**  
   By contributing, you agree that submitted documentation becomes part of the  
   ATLP public documentation layer and does not grant rights to any private logic.
